# Get Next Line - 42/Unit Factory

This project creates additional function that returns get next lines to '\n' from read's files.

Use my main.c and sample files to understand, how it is work.

1. Use "make" or "make re" in libft directory.
2. Compile with gcc -Wall -Wextra -Werror main.c get_next_file.c libft/libft.a
3. Run aplication ./a.out exemples/[choose_file_1] exemples/[choose_file_2] ...

Bonus:
• get_next_line has a single static variable.
• get_next_line may to manage multiple file descriptor.
